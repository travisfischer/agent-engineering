# Brainstorming

This skill provides process knowledge for clarifying what to build before deciding how to build it.

## Hard Constraints

1. This skill is brainstorming-only.
2. Do not implement code while running this skill.
3. Always produce a brainstorm artifact before handoff.

## Core Process

1. Assess whether brainstorming is needed.
2. Ask questions one at a time.
3. Explore 2-3 approaches with honest trade-offs.
4. Capture decisions in a structured brainstorm document.
5. Stop unless the user explicitly requests the next phase.
