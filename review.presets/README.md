# Review Presets

These files are ready-made starting points for `review.local.md`.

## Usage

1. Pick a preset from this directory.
2. Copy it to the host repo root as `review.local.md`.
3. Adjust `review_context` if needed.
4. Run the review workflow.

## Included Presets

1. `all-principles.md`
2. `minimal.md`
3. `quality-and-security.md`
4. `llm-systems.md`
