# Compound Engineering

Each unit of work should make future work easier, safer, and faster.

## Core Principle

Code is not the only output of engineering work. Plans, reviews, patterns, tests, evals, and written learnings are first-class artifacts.

## Operating Loop

`Plan -> Work -> Review -> Compound -> Repeat`

1. **Plan**
Clarify requirements and define a strong approach.

2. **Work**
Execute with continuous validation.

3. **Review**
Assess quality, risks, and regressions.

4. **Compound**
Capture reusable lessons and strengthen the system.

## Required Artifacts

1. A reusable plan
2. Review output with findings and rationale
3. Codified learnings
4. Updated checks that prevent repeat failures
