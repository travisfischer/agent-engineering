# Core Principles

These documents define the foundational principles for an agent-enabled software project.

Use them as a project constitution:

1. Treat them as direction-setting defaults
2. Apply them during planning, implementation, and review
3. Keep them high-level and durable
4. Move implementation detail into workflow docs rather than principle docs
