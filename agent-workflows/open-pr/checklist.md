# Open PR Checklist

1. Preflight checks passed.
2. Working state was preserved safely if needed.
3. Latest base branch was fetched.
4. Branch decision followed project policy.
5. Rebase was attempted and stopped on conflict when applicable.
6. Commit grouping stayed focused.
7. Required checks were run.
8. PR body includes the required sections.
9. Branch was pushed and PR was opened.
10. Execution summary was returned.
