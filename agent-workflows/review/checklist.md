# Review Checklist

1. Target was normalized correctly.
2. Reviewer registry loaded successfully.
3. Conditional triggers were evaluated.
4. One reviewer run was dispatched per selected reviewer.
5. Reviewer outputs were written outside the repo tree during execution.
6. Orchestrator synthesized all outputs into one persisted review artifact.
7. Temporary reviewer files were cleaned up automatically.
8. Summary includes all actionable findings.
