# PR Feedback Checklist

1. PR target was resolved correctly.
2. Unresolved review feedback was fetched.
3. Every in-scope feedback item was assessed.
4. Valid feedback was fixed with focused changes.
5. Non-actioned feedback received a clear response.
6. Changes were committed in logical units.
7. Branch was pushed after changes.
8. Final summary includes counts, actions, and blockers.
